一个非官方的 API python 玩具盒  
即将支持官方 API 👀

> *prefer python3.7+*  
> 欢迎 Star 🌟、Fork 🍴、Issue 💬、PR. 一起让 flomo 用的更加得心应手

## Usage 使用

```shell
pip install -U flomo
```

```python
import flomo
client = flomo.Flomo(api='https://flomoapp.com/xxxxxAPIxxxx')
client.new('hello flomo')
```
